# Esolang.Processor.Extensions.IO

Provides extension methods for running `IEventProcessor` using various I/O abstractions.

## Installation

```bash
dotnet add package Esolang.Processor.Extensions.IO
```

## Overview

This package provides extension methods to facilitate running processors (interpreters) based on an event-driven I/O model, supporting `TextReader`/`TextWriter`, `string` input/output, and `PipeReader`/`PipeWriter` from `System.IO.Pipelines`.
